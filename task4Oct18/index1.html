Java Full Stack Program

State:Kerala;
Public Partnerships:Kudumbashree, DDU GKY

Apex has collaborated with the Kudumbashree & DDU GKY to provide Java Full stack training for students from Kerala. This 6-month program is a fully residential training program, where students will be part of a 4-month classroom training followed by 2 month internship. Post which students would be provided with on-campus placement opportunities. FACE already has tie-ups with various enterprises to provide up to 300 software developers. They will be sourced from this training pool. The average package for this requirement is Rs. 3.5LPA
